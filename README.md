# 📊 Election Data Analysis & Machine Learning Project

## 🔍 Overview
This project performs Exploratory Data Analysis (EDA) and Machine Learning on an election dataset to uncover patterns in voting behavior and predict outcomes.

## 📌 Features
- Data Cleaning & Preprocessing
- Data Visualization (Matplotlib, Seaborn)
- Linear Regression (Predict vote share %)
- Logistic Regression (Predict winner)

## 📊 Visualizations
- Histogram
- Pie Chart
- Scatter Plot
- Heatmap
- Box Plot & More

## 🤖 Machine Learning Models
- Linear Regression
- Logistic Regression

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📈 Results
- Identified voting trends and party performance
- Built predictive models for vote share and winner classification

## 📁 Dataset
Election dataset with candidate details, vote share, turnout, etc.
